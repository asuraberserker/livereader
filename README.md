# Live Reader

一款 Windows 桌面工具，监听目标游戏或应用程序窗口内的鼠标点击事件，对指定屏幕区域截图并进行 OCR 文字识别，最后通过系统语音引擎朗读识别到的文字。

适合用于视觉小说、带字幕的游戏，或任何需要将屏幕文字转为语音播报的场景，无需对目标程序做任何修改。

---

## 功能特点

- **点击触发截图** — 不轮询，只在你点击目标窗口时才执行 OCR，几乎不占用空闲时的 CPU
- **PP-OCRv5 识别** — 基于 PaddleOCR 3.x，使用 PP-OCRv5 mobile 模型，精度与速度平衡
- **多语言支持** — 简体中文、繁体中文、英文、日文、韩文
- **灵活划定范围** — 拖拽选择字幕区域一次即可，设置自动保存，下次启动无需重新设置
- **读音替换** — 通过独立配置工具纠正常见误读字或调整 TTS 发音
- **深色界面** — 简洁的深色主题 UI

---

## 系统要求

- Windows 10 / 11（64 位）
- Python 3.10 – 3.12
- PaddlePaddle **3.2.0** CPU 版（其他版本在 Windows CPU 下存在已知 bug）

---

## 安装

```bash
# 1. 安装 PaddlePaddle CPU 版（版本必须是 3.2.0）
pip install paddlepaddle==3.2.0

# 2. 安装其余依赖
pip install -r requirements.txt
```

> **注意：** 不要将 `paddlepaddle` 升级到 3.2.0 以上版本。3.3.0 在 Windows CPU 下存在已确认的 oneDNN bug，会导致推理失败。

## 文件结构

```
livereader.py         主程序
range_selector.pyw    范围选择工具（由主程序启动）
words_config.pyw      读音替换配置工具
requirements.txt      Python 依赖列表
words.txt             读音替换规则（首次保存后自动创建）
capture_range.txt     已保存的截图范围（划定范围后自动创建）
last_process.txt      上次选择的进程名（自动创建）
```

## 使用方法

### 1. 启动主程序

```bash
python livereader.pyw
```

### 2. 选择目标窗口

从下拉列表中选择游戏或应用程序窗口。如果目标窗口不在列表中，点击 **刷新列表**。

### 3. 划定截图范围

点击 **划定范围**，屏幕会变暗并出现十字光标。拖拽鼠标框选字幕或对话文字所在区域，松开鼠标即完成确认，坐标会自动回报给主程序并保存。

### 4. 开始捕获

点击 **开始捕获**，主程序安装全局鼠标监听器。

此后，每次在目标窗口内点击左键，都会触发以下流程：

```
点击 → 等待 1 秒 → 截图 → OCR 识别 → 朗读
```

等待 1 秒是为了让游戏有足够时间渲染下一行文字。

### 5. 停止捕获

点击 **停止捕获**，移除鼠标监听器并停止语音播报。

## 读音替换

点击 **配置读音** 打开替换词编辑器。

每条规则将一个字符串映射为另一个字符串，适合纠正 OCR 误读的字符或调整 TTS 对特定词的发音。点击 **保存并通知主程序**，更改立即在主程序中生效，无需重启。

规则保存在 `words.txt`，格式为：

```
原词=>替换词
```

---

## 程序间通信（IPC）

三个程序通过 `multiprocessing.connection` 在本机 `127.0.0.1:6123` 建立 TCP 连接通信，使用共享密钥认证。主程序作为服务端，两个子工具连接后发送一条消息即退出。

| 消息类型 | 发送方 | 内容 |
|---|---|---|
| `range` | range_selector.pyw | `(x, y, width, height)` 绝对屏幕坐标 |
| `words` | words_config.pyw | `(原词, 替换词)` 元组列表 |

## 常见问题

**OCR 识别到 0 行文字**
请确认截图范围内确实有文字。PP-OCRv5 模型基于自然彩色图像训练，请避免选取纯白或对比度异常的区域。

**没有语音播出**
程序使用 Windows SAPI（`SAPI.SpVoice`）。请确保 Windows 设置 → 时间和语言 → 语音 中已安装至少一个 TTS 语音包。

**鼠标监听器安装失败**
如果出现 `pynput` 相关错误，请确认已安装 `pynput`，并检查杀毒软件是否阻止了全局输入钩子。

**范围选择器显示的屏幕不对**
多显示器环境下，蒙版覆盖所有显示器。保存的坐标为绝对屏幕坐标，无论游戏运行在哪块显示器上均可正常工作。
